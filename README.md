# .github

This repo is "template" files that are used by GitHub, when corresponding files
in the individual repos don't exist.

This contains:

- Code of Conduct -- used on other repos if an overriding file is not provided.
- Logo Font template

  Logos are built using this font, and then often displayed at 4x.
